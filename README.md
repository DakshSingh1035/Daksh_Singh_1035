Fake News Detection 📰

This is a simple machine learning project where I tried to classify news as real or fake using basic NLP techniques.

---

About the project

I built this project to understand how machine learning can be used to detect misinformation. It takes news text as input and predicts whether it is fake or real.

---

What I used

* Python
* Pandas
* Scikit-learn
* TF-IDF Vectorizer

---

How it works

* Clean the text data
* Convert text into numbers using TF-IDF
* Train a model (like Logistic Regression / Naive Bayes)
* Predict output based on input text

---

How to run

1. Clone the repo
2. Open the notebook
3. Run all cells

---

Example

Input:
"Aliens landed in India yesterday"

Output:
Fake
